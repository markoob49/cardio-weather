# КардиоПогода / CardioWeather 🫀🌤️

Персональное веб-приложение для мониторинга артериального давления с анализом влияния погодных факторов.

**→ [Открыть приложение](https://markoob49.github.io/cardio-weather/)**

---

## Что делает приложение

- Записывает измерения АД (САД/ДАД), ЧСС, SpO₂
- Автоматически прикрепляет погодные данные к каждому измерению (температура, атмосферное давление, влажность, PM2.5, Kp-индекс)
- Вычисляет индекс кардиориска на основе текущих метеоусловий
- Выявляет персональные погодные триггеры (корреляционный анализ Спирмена)
- Ведёт дневник самочувствия
- Генерирует отчёт для врача в формате ESH/ESC
- Поддерживает ввод данных задним числом (до 30 дней) с автоподгрузкой исторической погоды

## Особенности

- **Все данные хранятся локально** на устройстве пользователя (localStorage). На сервер ничего не передаётся.
- Работает как PWA — можно добавить на главный экран смартфона
- Двуязычный интерфейс: русский и иврит
- Не требует регистрации и установки

## Источники данных

| Данные | Источник |
|---|---|
| Погода (текущая и архив) | [Open-Meteo](https://open-meteo.com/) |
| PM2.5 / PM10 | [Open-Meteo Air Quality](https://open-meteo.com/) |
| Kp-индекс (геомагнитная активность) | [NOAA SWPC](https://www.swpc.noaa.gov/) |
| Геокодирование | [Open-Meteo Geocoding](https://open-meteo.com/) |

Все API бесплатны и не требуют ключей.

## Медицинский дисклеймер

Приложение является персональным инструментом самомониторинга и **не является медицинским устройством**. Не предназначено для постановки диагноза или замены консультации врача. При ухудшении самочувствия обращайтесь к врачу.
